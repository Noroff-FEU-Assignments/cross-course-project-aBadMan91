Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

I would start to tell them to do some research on threats like "Cross-Site scripting (XSS)", "SQL Injection", "Cross-Site Request Forgery (CSRF)" and other threats like "Clickjacking, Denial of Service, Directory Traversal, File Inclusion and Command Injection".

Then I would make this small list:

* Make sure that the «HTTPS» and «HSTS» is always used on your website.

* Use better password management, the user should have to use letters like “!?#” in the password and numbers. The stronger the password the better. Make it possible to use two-factor authentication.

* Tracking the most known or “popular” threats is important, they can be found on https://owasp.org/www-project-top-ten/

* You should use vulnerability scanning tools to check the security of your page. One example of those tools are https://pentest-tools.com/, you can find more on https://owasp.org/www-community/Vulnerability_Scanning_Tools.

* You should only store data that you need. If you store sensitive data like a user’s credit card information you should never display the whole number, only the 4 last digits of the card should be visible so the user can still recognize the card information

* You should always check and sanitize all data.
